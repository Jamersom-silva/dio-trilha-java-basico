# ContaBanco

Projeto desenvolvido durante o curso "Trilha Java Básico" da DIO.

## 📋 Descrição
Sistema simples para criação de conta bancária via terminal, coletando dados do usuário e exibindo uma mensagem de confirmação.

## 🚀 Funcionalidades
- Coleta número da agência
- Coleta número da conta
- Coleta nome do cliente
- Coleta saldo inicial
- Exibe mensagem personalizada com os dados inseridos

## 🛠️ Tecnologias
- Java
- Scanner (para entrada de dados via terminal)

## 📦 Como executar
1. Clone o repositório
2. Navegue até a pasta src
3. Compile: `javac ContaTerminal.java`
4. Execute: `java ContaTerminal`

## 📝 Exemplo de uso
Por favor, digite o número da Agência !
067-8
Por favor, digite o número da Conta !
1021
Por favor, digite o nome do Cliente !
MARIO ANDRADE
Por favor, digite o saldo da Conta !
237.48

Olá MARIO ANDRADE, obrigado por criar uma conta em nosso banco,
sua agência é 067-8, conta 1021 e seu saldo 237.48 já está disponível para saque.
